# 📞 Contact Manager Web App

A simple and interactive **Contact Manager application** built using **HTML, CSS, and JavaScript**.  
This app allows users to add, view, and delete contacts easily in a structured table format.

---

## 🚀 Features
- ➕ Add new contacts (Name, Phone, Email)  
- 📋 Display contacts in a table  
- ❌ Delete contacts  
- 🎨 Clean and user-friendly UI  
- ⚡ Instant updates (no page reload)  
- 📱 Responsive layout  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## ▶️ How to Run
1. Download or clone this repository  
2. Open **contact.html** in your browser  

---

## 📸 Overview
This app allows you to:
- Enter contact details (Name, Phone, Email)  
- Add them to a table  
- View all saved contacts  
- Delete contacts when needed
